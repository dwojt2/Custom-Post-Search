A plugin with a custom search form. It also displays a post based on search results.

== Description ==

This plugin creates a custom search form that allows a user to search a specific post type and view a post with the same title as the search query. 

Shortcode Style:
* [post_search] - will search any post type
* [post_search post_type="page"] - will search the post type specified in the attribute. The settings page of this plugin will list all available post types.
